🏥 Hospital Management System – SQL Data Analytics Project
📌 Overview

This project focuses on designing and analyzing a Hospital Management System database using SQL. It includes the creation of relational tables, insertion of sample data, and analytical queries to explore hospital operations such as patient records, doctors, nurses, appointments, billing, and departmental information.

The project demonstrates practical SQL concepts used in real-world healthcare data analysis.

🚀 Features
Database creation using SQL
Relational database design with Primary Keys and Foreign Keys
Data insertion and management
Complex SQL queries for data analysis
Hospital operational insights through SQL
Data integrity using constraints

🛠 Technologies Used
MySQL
SQL Workbench

📊 SQL Concepts Covered
DDL (CREATE, ALTER, DROP)
DML (INSERT, UPDATE, DELETE)
SELECT Queries
JOINs
Aggregate Functions
GROUP BY & HAVING
ORDER BY
Subqueries
Constraints (Primary Key & Foreign Key)

📁 Database Includes
The database contains tables related to:
Departments
Doctors
Nurses
Patients
Rooms
Appointments
Staff
Billing
Medical Records
And other hospital-related entities

🎯 Learning Outcomes
Database schema design
Relational database management
Writing optimized SQL queries
Performing healthcare data analysis
Understanding relationships between multiple tables
